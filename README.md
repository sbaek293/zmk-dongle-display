# zmk-dongle-display

A ZMK dongle with a 1.69-inch flat display (ST7789V, 240×280).

## Case / Enclosure

Printable STEP and STL case files for the 1.69-inch display module are in the [`case/`](case/) directory.

See [`case/README.md`](case/README.md) for dimensions, assembly instructions, and how to regenerate the files.
